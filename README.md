## üí° Core Learning Objectives
This project demonstrates practical mastery over the following concepts:
- Container Fundamentals: Pulling, running, and managing the lifecycle of single containers.
- Image Creation: Authoring custom Dockerfiles to package applications and solve the "it works on my machine" problem.
- Data Persistence: Safeguarding application state using Docker Volumes for persistent data storage (e.g., databases).
- Development Workflow: Utilizing Bind Mounts and tools like nodemon to create efficient, live-reloading local development environments.
- Networking: Creating custom Docker Networks to enable secure communication between application and database containers.
- Orchestration: Using Docker Compose to define, launch, and manage entire multi-container application stacks with a single, declarative file (docker-compose.yml).
- Image Optimization: Implementing Multi-Stage Builds to produce lean, production-ready images free of development dependencies.
- Security & Secrets: Remediating hardcoded secrets by leveraging .env files and the env_file directive in Docker Compose.
